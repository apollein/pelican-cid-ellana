# Cid

Cid is a simple theme for static site generator, [Pelican][pelican]

## Features:
* Disqus comment. `DISQUS_SITENAME` must be defined in configuration file.
* Google Analytics. `GOOGLE_ANALYTICS` must be defined in configuration.
* Responsive theme.
* Landing.
* Pagination.
